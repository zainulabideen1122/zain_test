Ecode Finder

Intro:


E-CODE Finder is a Full-Stack Website. Muslims are living in different countries of the world. The main problem of the muslims is to distinguish between Halal and Haram(Forbidden). So, government of Uk and of many countries decided to introduce ecode system. Ecode is a special code which is attach on the back of every product. By seeing this code, user could easily find out the status of the product. This website stores all ecodes and tell the user that which code is Halal , Haram or Mushbooh(Not confirmed)



Technologies:


In this website, I used HTML /CSS /JavaScript and Vue.js(Vue /Vue Routers /Vuex) in Front-End and Flask(Python) in Backend and MySQL in Database.
